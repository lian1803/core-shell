# 트렌드 분석: 네이버 플레이스 자동 진단 + PPT 자동 생성

## 관련 트렌드

1. **No-Code/Low-Code 자동화 도구의 B2B2C 확산** ⭐
   - Make, Zapier 같은 워크플로우 자동화가 마케팅 에이전시로 확대 중
   - 영업 프로세스 자동화가 핵심 경쟁력

2. **AI 기반 데이터 시각화 + 인사이트 생성** ⭐
   - ChatGPT, Claude로 분석 데이터를 자동으로 인사이트로 변환
   - PPT 자동 생성 도구(Beautiful.ai, Gamma.app) 성장

3. **지역 기반 SEO/로컬 검색 최적화 붐** ⭐
   - 구글 로컬 서비스, 네이버 플레이스 최적화가 소상공인 마케팅 핵심
   - "지도 SEO" 대행이 새로운 수익처

4. **마케팅 에이전시의 자동화 솔루션 자체가 판매 상품화**
   - 에이전시 자체 도구 → SaaS화 하는 트렌드 증가

## 리안에게 의미 있는 것

**"개인이 에이전시급 제안서를 30초에 만드는" 차이점이 곧 경쟁력.** 지금 이 영역은 손으로 하는 곳이 대부분인데, 자동화 + 데이터 기반 인사이트 조합이 시장을 재편할 가능성 높음.

## 활용 제안

1. **MVP 검증 우선**: 실제 에이전시 5곳에 쓰게 하고 "이 PPT로 계약 받았나?" 물어보기
2. **수익화 모델**: 
   - B2B2C: 에이전시에 월 구독 (PPT 1매 당 과금도 가능)
   - B2C 직접: 소상공인 직판 (자가진단 무료 → 상담 전환)
3. **차별점 강화**: 
   - 네이버 플레이스 + 구글 비즈니스, 당근마켓까지 멀티 크롤링
   - PPT에 "우리와 비교했을 때 경쟁사는?" 벤치마킹 페이지 추가
4. **재사용성**: 생성된 PPT → 에이전시가 고객에게 쓰면서 자기 브랜드 임베드되도록 설계
