# PRD v1.0 — 소상공인 인스타그램 자동화 서비스

> 작성자: 지훈 (실행팀 PRD)
> 기준: 준혁 이사 조건부 GO 판단 반영
> MVP 범위: **카페 업종 단일**, 자동 생성 + 원탭 발행 모델 (Meta API PoC 결과 반영)

---

## 제품 개요

카페 사장님이 업종 정보만 입력하면, 매일 인스타그램 캡션+이미지를 자동 생성해 원탭으로 발행할 수 있는 월 9,900원 구독 서비스

---

## Must Have

1. **업종 온보딩**: 업종(카페 고정), 가게 이름, 분위기 키워드(3개 선택), 대표 메뉴 입력
2. **일일 콘텐츠 자동 생성**: 매일 오전 8시, AI가 캡션(본문 + 해시태그 10개) + 정방형 이미지(1:1) 1세트 자동 생성
3. **콘텐츠 미리보기 + 수정**: 생성된 캡션/이미지를 발행 전에 확인하고 텍스트 직접 수정 가능
4. **원탭 발행**: "인스타에 올리기" 버튼 1번으로 Instagram Graph API를 통해 발행 (사용자 승인 트리거 방식)
5. **발행 예약**: 원하는 시간으로 예약 발행 설정 가능 (당일 기준, 최대 23:59까지)
6. **발행 히스토리**: 지난 30일 발행된 콘텐츠 목록 및 인스타 좋아요/도달 수 기본 조회
7. **구독 결제**: 월 9,900원 카드 자동결제, 첫 30일 무료 체험

---

## Must NOT (범위 외)

- ❌ 완전 자동 발행 (사용자 승인 없는 자동 포스팅) — Meta API 정책상 불가, MVP에서 제외
- ❌ 릴스(Reels) 생성 및 발행 — 기술 난이도 높음, 프리미엄 플랜 이후 검토
- ❌ DM 자동화 — 동일, 프리미엄 이후
- ❌ 카페 외 업종 (네일샵, 식당 등) — PMF 확인 후 확장
- ❌ 멀티 계정 관리 — 1계정 1구독 고정
- ❌ 해시태그 성과 분석 리포트 — 프리미엄 플랜 기능
- ❌ 이미지 커스텀 디자인 편집기 — 생성된 이미지 교체(재생성)만 허용
- ❌ 팀원 초대 / 협업 기능

---

## User Flow

```
[최초 가입 ~ 첫 발행]

1단계: 랜딩 페이지 접속 → "30일 무료 시작" CTA 클릭

2단계: 회원가입 (이메일 + 비밀번호) → 카드 정보 입력 (30일 후 자동 과금 안내)

3단계: 온보딩 — 업종 확인(카페 고정) → 가게 이름 입력
        → 분위기 키워드 선택 (예: 아늑한 / 트렌디 / 감성적 / 모던 / 편안한 중 3개)
        → 대표 메뉴 2~3개 입력 (예: 아메리카노, 크로플)

4단계: Instagram 계정 연동
        → "인스타그램 연결하기" 클릭
        → Instagram OAuth 인증 팝업 → 계정 권한 승인
        → 연동 완료 확인 화면

5단계: 첫 번째 콘텐츠 즉시 생성
        → "오늘의 콘텐츠 생성하기" 클릭
        → 로딩 (AI 생성 중, 예상 10~20초)
        → 캡션 + 이미지 미리보기 화면 노출

6단계: 콘텐츠 확인 및 수정
        → 캡션 텍스트 직접 수정 가능
        → 마음에 안 들면 "이미지 다시 생성" 또는 "캡션 다시 쓰기" 클릭 (1회 추가 생성 허용)

7단계: 발행
        → "지금 바로 올리기" 또는 "예약 발행" 선택
        → 예약 시: 시간 선택 (오늘 날짜 기준)
        → 최종 확인 → "인스타에 올리기" 버튼 클릭
        → Instagram Graph API 호출 → 발행 완료 토스트 메시지

[이후 매일 루틴]

8단계: 매일 오전 8시 — 앱 푸시 알림 또는 이메일 발송
        → "오늘의 콘텐츠가 준비됐어요 👀" 알림

9단계: 사용자 앱 접속 → 미리보기 확인 → 수정 or 원탭 발행

10단계: 히스토리 탭에서 지난 발행 콘텐츠 및 인스타 기본 지표 확인
```

## 화면 목록

| 화면명 | 설명 | 핵심 요소 |
|--------|------|-----------|
| **랜딩 페이지** | 서비스 소개 및 전환 유도 | 헤드카피, 기능 3줄 요약, "30일 무료 시작" CTA 버튼, 요금제 안내 |
| **회원가입 / 로그인** | 이메일 기반 인증 | 이메일 입력, 비밀번호 설정, 구글 소셜 로그인 옵션, 카드 정보 입력(Stripe/토스페이먼츠) |
| **온보딩 — 가게 설정** | 최초 1회, 업종 정보 수집 | 가게 이름 텍스트 필드, 분위기 키워드 선택 칩(5개 중 3개), 대표 메뉴 입력 필드(최대 3개), 다음 버튼 |
| **온보딩 — 인스타 연동** | Instagram OAuth 연결 | "인스타그램 연결하기" 버튼, 연동 성공/실패 상태 표시, 연동된 계정명 확인 |
| **홈 (오늘의 콘텐츠)** | 메인 화면, 매일 진입점 | 생성된 이미지 썸네일, 캡션 미리보기(접기/펼치기), "다시 생성" 버튼, "지금 올리기" / "예약 발행" 버튼, 남은 무료 체험 일수 배지 |
| **콘텐츠 수정** | 캡션 직접 편집 | 캡션 텍스트 에디터, 해시태그 별도 영역, 글자 수 카운터(2,200자 제한), 저장 버튼 |
| **예약 발행 설정** | 발행 시간 선택 | 시간 피커(오늘 기준 현재 시간 이후), 예약 확정 버튼, 예약 취소 옵션 |
| **발행 완료** | 발행 후 피드백 화면 | 성공 애니메이션, 인스타 게시물 바로가기 링크, "히스토리 보기" 링크 |
| **히스토리** | 지난 30일 발행 기록 | 발행 날짜순 목록(이미지 썸네일 + 캡션 첫 줄), 인스타 좋아요 수 / 도달 수 표시, 클릭 시 인스타 원본 게시물 이동 |
