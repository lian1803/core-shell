## 포지셔닝

**"GPT 켜고, 캔바 켜고, 인스타 켜는 그 삼단 루틴 — 이제 버튼 하나로"**

차별점: 기존 예약 발행툴(버퍼, 나중에)은 콘텐츠를 **직접 만들어야** 함. 우리는 만들기부터 올리기까지 **원스톱**. 소상공인 특화라 일반 AI 툴보다 업종 맥락이 맞음.

## 카피 A/B

### A안 — 손실회피 프레이밍
```
제목: 오늘도 인스타 못 올렸죠?
서브: 카페 사장님 하루 평균 37분을 SNS 콘텐츠에 씁니다.
      업종만 입력하면, 오늘부터 그 37분을 돌려드립니다.
CTA: 30일 무료로 되찾기 →
```

### B안 — 앵커링 + 사회적 증거
```
제목: 마케터 월급 300만원 vs 월 9,900원
서브: AI가 매일 오전 8시, 카페 맞춤 사진과 글을 만들어 인스타에 올려줍니다.
      첫 달은 무료입니다.
CTA: 지금 무료로 시작하기 →
```

> **추천 우선 집행**: B안으로 시작, 클릭률 낮으면 A안 교체
> 검증 기준: CTR 3% 이상이면 카피 유지

## 채널 전략

| 채널 | 형식 | 빈도 | 목표 |
|------|------|------|------|
| 카카오 오픈채팅 (소상공인 카페·식당 방 침투) | 텍스트 후기형 게시글 | 주 2회 | 초기 50명 유입 |
| 네이버 카페 (자영업자 커뮤니티, 아프리카·동네 카페사장) | 정보글 위장 후기 + CTA | 주 2회 | SEO + 직접 유입 |
| 인스타그램 (본 계정) | 비포/애프터 릴스, 카드뉴스 | 주 3회 | 팔로워 확보 + 신뢰 구축 |
| 유튜브 쇼츠 | "카페 사장 하루 브이로그" 스타일 | 주 1회 | 알고리즘 노출 |
| 당근마켓 동네홍보 | "우리 동네 카페 사장님께" 직접 공지 | 런치 주 매일 | 로컬 0원 획득 |
| 스레드(Threads) | 날것의 빌딩 인 퍼블릭 | 매일 | 팔로워 없어도 확산 가능 |

## 48시간 검증 플랜

### Day 1 (오늘)

- [ ] **오전**: 노션or 구글폼으로 랜딩 페이지 대용 페이지 만들기 (Carrd.co 무료 플랜 — 30분 완성)
  - 헤드카피 B안 적용, 이메일 수집 폼만 있으면 됨
- [ ] **오후 1시**: 카카오 오픈채팅 "카페 사장님 모임" 5개 방 찾아서 가입
- [ ] **오후 3시**: 아래 초기 게시물 초안 스레드 + 카카오 오픈채팅에 동시 발행
- [ ] **저녁**: 네이버 카페 "자영업자/카페창업" 커뮤니티 2곳에 정보글 형식으로 게시

**Day 1 목표**: 랜딩 이메일 수집 **20명**

---

### Day 2 (내일)

- [ ] **오전**: 어제 반응 집계 (댓글 수, 링크 클릭, 이메일 수집 수)
- [ ] **오전**: 반응 있는 채널에 A안 카피로 교체 테스트
- [ ] **오후**: 이메일 수집된 분들한테 직접 카카오톡 or 이메일로 "베타 사용자 인터뷰 제안" DM
  - "5분만 얘기 나눌 수 있을까요? 무료 3개월 드릴게요"
- [ ] **저녁**: 당근마켓 내 동네 + 인근 상권 3곳에 동네홍보 게시

**Day 2 목표**: 인터뷰 약속 **5건**, 추가 이메일 **20명** (누적 40명)

---

**최종 검증 기준**

```
✅ PASS: 48시간 내 이메일 수집 40명 이상 → 채널 확대 집행
⚠️ 재검토: 20~39명 → 카피 교체 후 재시도
❌ 피벗 필요: 20명 미만 → 타겟 or 메시지 재설정
```

## 바이럴 루프

```
[유입]
당근/오픈채팅/스레드에서 "이거 진짜 편하겠다" 반응 유도
↓
[활성화]
30일 무료 체험 → 온보딩 3분 완료 → 첫 콘텐츠 생성 즉시 경험
"이게 되네?" 첫 번째 아하 모먼트 확보
↓
[공유 유도 — 핵심]
발행 완료 화면에서:
"이 게시물은 [브랜드명]으로 만들었어요 🤖" 자동 워터마크 or 댓글 삽입
→ 사장님 팔로워들이 "이거 어떻게 했어요?" 질문
→ 사장님이 자연스럽게 공유
+
사용 후기 작성 시 → 다음 달 50% 할인 쿠폰 (비용 0원, 구독료 절반)
↓
[추천]
"친구 카페 사장님 소개하면 둘 다 한 달 무료"
→ 소상공인은 동업자 네트워크가 강함 (같은 상권, 업종 커뮤니티)
↓
[반복]
매일 오전 8시 푸시알림 = 매일 앱 재진입
히스토리 탭 좋아요/도달 수치 = 성과 확인 → 지속 동기부여
```
