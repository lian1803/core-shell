# PRD v1.0 — 마케터-고객 실시간 채팅 + 채팅 내 결제 링크 연동

**작성자**: 지훈 (실행팀 PRD)
**작성일**: 2025-01-27
**버전**: MVP v1.0
**상태**: 개발 착수 가능

---

## 제품 개요

B2B 마케터가 고객과 실시간 채팅하면서 카카오페이·토스 결제 링크를 채팅창 안에서 바로 생성·발송해 이탈 없이 구매 전환을 완결하는 웹 기반 채팅 솔루션.

---

## Must Have

1. **마케터 계정 관리** — 회원가입·로그인·워크스페이스 생성 (이메일 + 비밀번호)
2. **채팅 위젯 발급 및 삽입** — 마케터가 위젯 스크립트(JS embed code)를 발급받아 자사 사이트에 붙일 수 있음
3. **실시간 1:1 채팅** — 마케터(대시보드) ↔ 고객(위젯) 간 WebSocket 기반 양방향 메시지 송수신
4. **채팅 내 결제 링크 생성 및 발송** — 마케터가 상품명·금액을 입력 → 카카오페이 또는 토스페이먼츠 결제 URL 생성 → 채팅창에 버블 형태로 발송
5. **결제 상태 실시간 추적** — 결제 링크 발송 후 고객의 결제 완료·실패·취소 상태를 채팅창 내 뱃지로 즉시 표시
6. **기본 전환 대시보드** — 채팅 건수·결제 링크 발송 수·결제 완료 수·전환율을 날짜별로 조회
7. **대화 히스토리 보관** — 채팅 종료 후 대화 내용 저장 및 마케터 대시보드에서 조회

---

## Must NOT (범위 외)

- CRM 통합 및 고객 데이터 연동 (Phase 2)
- AI 챗봇·자동 응답·AI 분석 기능 (Phase 2)
- 카카오톡·라인·인스타그램 등 외부 메신저 채널 연동 (Phase 2)
- 마케터 팀 계정·권한 관리 (멀티 에이전트) (Phase 2)
- 모바일 앱 (iOS/Android) (Phase 2)
- 자체 PG 정산 시스템 — 결제는 카카오페이·토스페이먼츠 API로만 처리, 자체 정산 로직 없음
- 환불·부분취소 처리 UI (Phase 2)
- 결제 링크 외 파일 전송·이미지 첨부 (Phase 2)
- 다국어 지원

## User Flow

### Flow A. 마케터 온보딩 (최초 1회)

```
1단계: 회원가입 페이지 접속 → 이메일·비밀번호·회사명 입력 후 가입
2단계: 이메일 인증 링크 클릭 → 로그인
3단계: 워크스페이스 이름 설정 (예: "리안 쇼핑몰")
4단계: 위젯 스크립트 발급 페이지에서 JS embed code 복사
5단계: 자사 사이트 <body> 태그 안에 스크립트 붙여넣기 → 위젯 노출 확인
6단계: PG 연동 설정 탭 → 카카오페이 또는 토스페이먼츠 API Key 입력·저장
       (※ PG사 계약·API Key 발급은 사전 완료 전제)
```

### Flow B. 실시간 채팅 + 결제 링크 발송 (핵심 플로우)

```
1단계: 고객이 마케터 사이트에서 위젯 채팅 버튼 클릭 → 이름·이메일 입력 후 채팅 시작
2단계: 마케터 대시보드에 새 채팅 알림(브라우저 알림 + 빨간 뱃지) 표시
3단계: 마케터가 대화창 클릭 → 고객 메시지 확인 후 텍스트 응답
4단계: 마케터가 결제 링크 생성 버튼(💳 아이콘) 클릭
5단계: 모달창에서 상품명·금액·메모(선택) 입력 → [링크 생성 및 발송] 클릭
6단계: 채팅창에 결제 버블 발송됨 (고객 화면에도 즉시 표시)
7단계: 고객이 결제 버블 내 [지금 결제하기] 버튼 클릭 → PG사 결제 페이지로 이동
8단계: 고객이 결제 완료 → 채팅창 내 결제 버블 상태가 "결제 완료 ✅"로 실시간 변경
       (결제 실패 시 "결제 실패 ❌" / 24시간 미결제 시 "만료" 표시)
9단계: 마케터·고객 모두 결제 완료 상태 확인 → 대화 종료
```

### Flow C. 전환 대시보드 조회

```
1단계: 마케터 로그인 → 대시보드 탭 클릭
2단계: 날짜 범위 선택 (기본값: 최근 7일)
3단계: 채팅 건수·링크 발송 수·결제 완료 수·전환율 카드 확인
4단계: 대화 히스토리 탭에서 과거 대화 검색·조회
```

|--------|------|-----------|
| 회원가입 | 마케터 계정 생성 | 이메일·비밀번호·회사명 입력 필드, [가입하기] 버튼, 이메일 인증 안내 문구 |
| 로그인 | 마케터 로그인 | 이메일·비밀번호 입력 필드, [로그인] 버튼, 비밀번호 찾기 링크 |
| 온보딩 — 위젯 설정 | 위젯 스크립트 발급 및 PG 연동 설정 | JS embed code 복사 버튼, PG사 선택(카카오페이/토스) 라디오, API Key 입력 필드, [저장] 버튼, 연동 성공/실패 상태 표시 |

| 마케터 대화창 | 고객과의 실시간 채팅 | 메시지 말풍선(좌: 고객, 우: 마케터), 텍스트 입력창, 전송 버튼, 결제 링크 생성 버튼(💳), 결제 버블 컴포넌트(상품명·금액·상태 뱃지·[재발송] 버튼) |
