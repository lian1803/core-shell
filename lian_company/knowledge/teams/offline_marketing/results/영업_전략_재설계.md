# 리안 컴퍼니 — 경기 북부 소상공인 네이버 플레이스 대행 영업 전략

---

## 핵심 인사이트 (자료 + 리서치 분석)

**인사이트 1 — 타겟의 진짜 적은 "바쁨"이 아니라 "모름"이다**
(출처: 한국 소상공인 Pain Point 리서치)
디지털 마케팅 역량 자기평가 41.1점, 음식점 38.1점 최저. "사진·메뉴 꼼꼼히 올렸는데 순위 안 변함"이 실제 언어다. 이들은 게으른 게 아니라 체류시간·외부유입 같은 로직 자체를 모른다. 영업 메시지는 "더 열심히 하세요"가 아니라 "모르는 게 당연해요, 제가 대신 합니다"여야 한다.

**인사이트 2 — 손실회피가 이득 제시보다 2배 강하다, 그런데 잘못 쓰면 역효과**
(출처: 손실회피 편향 리서치 + 대행사 거절 이유 Top 5)
"안 하면 망해요" 압박은 이미 소상공인이 학습된 사기 패턴으로 인식한다. 애견미용실 유 사장 사례가 증거. 손실회피는 써야 하되, **공포 유발이 아니라 데이터 기반 손실 계산**으로 제시해야 한다. "지금 순위에서 매달 [N]만 원 기회비용이 발생하고 있어요"가 "안 하면 손님 뺏겨요"보다 10배 더 신뢰를 얻는다.

**인사이트 3 — 방문자 1명 감소 = 매출 9,256원 손실이라는 숫자가 클로징 핵심 무기**
(출처: 네이버 플레이스 순위와 매출 상관관계 실증 데이터)
상관계수 0.90~0.92, R² 0.828~0.85. 이 숫자를 영업 현장에서 직접 쓴다. 방문자 수 차이를 돈으로 환산하는 순간 타겟은 추상적 "마케팅 필요성"에서 구체적 "지금 손해 금액"으로 전환된다. 이게 클로징 전환점이다.

**인사이트 4 — 경기 북부는 디지털 인식 격차가 오히려 기회다**
(출처: 경기 북부 디지털 마케팅 인식 수준 리서치)
디지털 광고 활용률 15.4%, 경기도 전체 43.3점. 경쟁 대행사가 서울·수도권 핵심부에 집중할 때 경기 북부(양주·포천·의정부)는 아직 대행사 포화 전이다. 진입 장벽이 낮고 레퍼런스 축적 후 지역 내 사회적 증거 효과가 폭발적으로 작동한다.

**인사이트 5 — 첫 DM에서 계약까지 신뢰 사다리를 3단계로 설계해야 한다**
(출처: Challenger Sale + SPIN Selling + 대행사 거절 이유 + 현재 영업 스크립트)
기존 스크립트의 문제: 콜드DM → 바로 계약 시도. 소상공인은 한 번 영업 사기 경험 후 즉시 계약 요청 = 사기 패턴으로 학습되어 있다. Challenger Sale의 "Teach → Tailor → Take Control" 구조를 적용, **무료 진단 리포트 → 문제 구체화 → 해결안 제시** 3단계로 신뢰를 쌓은 뒤 계약으로 간다.

## 영업 전략 설계

### 타겟 선별 기준

**1순위 타겟 — 이 3가지를 동시에 충족하는 업체만 공략한다**

```
① 네이버 플레이스 등록은 되어 있으나 최근 90일 내 게시물/소식 업데이트 없음
② 해당 지역 + 업종 키워드(예: "의정부 미용실") 검색 시 5위 이하
③ 리뷰 수 경쟁 상위 업체 대비 50% 미만 (상위 3개 업체 평균과 비교)
```

**업종별 우선순위:**
- **1순위: 미용실** — 재방문율·예약 구조상 신규 고객 유입이 매출에 직결. 원장이 직접 결정권 보유, 승인 단계 없음. 평균 시술 단가 3~8만 원으로 방문자 1명 증가 효과가 즉각적.
- **2순위: 식당** — 방문자-매출 상관관계 실증 데이터(R² 0.85)가 가장 강력. 손실 계산이 가장 쉬워 클로징 설득력 최고. 단, 점심·저녁 영업 중 연락 절대 금지.
- **3순위: 카페** — 단가 낮아 ROI 설득 난이도 높음. 미용실·식당 레퍼런스 확보 후 공략.

**타겟 발굴 방법 (매일 30분):**
1. 네이버 지도에서 "의정부 미용실", "양주 맛집", "포천 카페" 검색
2. 5위 이하 업체 중 최근 소식 없는 업체 리스트업
3. 카카오맵에서 동일 검색 → 네이버와 카카오 모두 노출 약한 업체 = 1순위

### 접촉 전략

#### STEP 1 — 콜드DM 발송 (첫 접촉)

**발송 시간: 오전 10:00~11:30 고정**
이유: 식당·카페 점심 영업 준비 중 12시 이후는 절대 안 됨. 미용실은 10시 오픈 직후 첫 고객 응대 전 확인 가능.

**실전 DM 스크립트 (버전 A — 데이터 충격 버전, 미용실용):**

```
사장님, 안녕하세요.
오늘 '의정부 미용실' 네이버 검색해봤는데
[업체명]이 지금 [N]위에 있더라고요.

1~3위 업체는 한 달에 검색 유입이 
사장님 업체보다 약 3~4배 많아요.

이거 방문자 수로 따지면 
매달 [N]만 원 정도 차이 날 수 있거든요.

무료로 정확한 분석 자료 만들어드렸어요.
부담 없이 받아보실래요?
```

**실전 DM 스크립트 (버전 B — 공감형, 식당용):**

```
사장님, 안녕하세요.
요즘 장사 괜찮으세요?

'포천 맛집' 검색하면 사장님 업체가 
지금 [N]위에 나오거든요.

근처에 비슷한 업체들이 1~3위 차지하고
거기로 손님이 다 가는 구조예요.

사장님 업체 무료로 분석해뒀는데
한번 확인해보시겠어요? 부담 없어요.
```

**발송 규칙:**
- 하루 최대 20건 (기존 25건에서 하향 — 개인화 품질 유지)
- 업체명, 지역, 순위 반드시 개인화. 공통 DM은 스팸으로 인식되어 차단
- 3일 후 무응답 시 1회 팔로업, 이후 완전 포기

**팔로업 DM:**

```
사장님, 며칠 전에 플레이스 분석 자료 
연락드렸던 리안이에요.

아직 확인 못 하셨을 것 같아서요.
분석 자료 만들어둔 게 있는데
보내드려도 될까요? 판단은 사장님이 하시면 돼요.
```

#### STEP 2 — 첫 응답 후 진단 리포트 전달 (신뢰 구축)

**응답이 오면 즉시 (30분 이내) 진단 리포트 전송**

진단 리포트에 반드시 포함할 항목:
1. 현재 순위 (스크린샷)
2. 상위 3개 경쟁 업체와 리뷰 수·사진 수·소식 업데이트 빈도 비교표
3. 해당 키워드 월간 검색량
4. **손실 계산식**: "현재 순위 vs 3위 진입 시 추정 방문자 차이 × 9,256원 = 월 기회비용"

**리포트 전달 후 멘트:**

```
사장님, 분석 자료 보내드렸어요.
보시고 궁금한 거 있으면 편하게 물어봐 주세요.

딱 한 가지만 말씀드리면,
지금 [N]위에서 3위로만 올라가도
월 방문자 [N]명 차이 나는데,
이게 매출로 계산하면 매달 [N
