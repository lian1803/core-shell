# 리안 컴퍼니 — 경기 북부 소상공인 네이버 플레이스 대행 영업 카피 완성본

---

## DM/문자 스크립트

### 1차 메시지 (3가지 버전)

**버전 A — 손실 언어 직격 (식당·고깃집 타겟 / Cold)**

```
안녕하세요 사장님 :)
지나가다 [업체명] 네이버 플레이스 잠깐 봤어요.

지금 '[의정부/양주/포천] [음식종류]' 검색하면
사장님 가게가 [N]위에 계시더라고요.

솔직히 말씀드리면—
1위 업체는 한 달에 [N]번 조회되고
사장님 가게는 [N]번이에요.

이 차이, 한 달 매출로 환산하면
약 [N]만원씩 매달 빠져나가고 있는 거거든요.

제가 개선 포인트 무료로 정리해뒀는데
한번 받아보시겠어요? 부담 없이 보내드릴게요 :)
```
→ 왜 이 문장: "매달 N만원 빠져나간다"는 손실 프레임이 이득 언어보다 2배 강하게 반응하는 식당 사장님 심리를 직격하고, 숫자 기반이라 영업 냄새 대신 진단 느낌을 준다.

---

**버전 B — 경쟁자 비교 자극 (미용실 타겟 / Cold)**

```
안녕하세요 사장님 :)
[업체명] 네이버 플레이스 보다가 연락드려요.

'[지역] 미용실' 검색해보니까
지금 [N]위에 계신데요,

1위 업체랑 딱 두 가지가 달라요.
→ 사진 수: 사장님 [N]장 / 1위 [N]장
→ 리뷰: 사장님 [N]개 / 1위 [N]개

이 두 개 차이로 신규 손님이
한 달에 [N]명씩 1위 쪽으로 넘어가는 구조예요.

바꾸면 달라지는 게 명확해서
무료로 자료 만들어놨어요. 받아보시겠어요? :)
```
→ 왜 이 문장: 미용실 사장님은 "내 가게가 왜 안 되는가"보다 "옆 가게가 왜 잘 되는가"에 더 민감하게 반응하므로, 구체적 수치 비교로 경쟁 자극을 건드린다.

---

**버전 C — 공감형 진입 (카페 타겟 / Cold)**

```
안녕하세요 사장님 :)
[업체명] 플레이스 보다가 연락드려요.

사진이 [N]장이고
마지막 리뷰가 [N]달 전이시더라고요.

네이버가 요즘 딱 이 두 가지로
순위를 가르거든요.
그래서 지금 '[지역] 카페' 검색하면
[N]위에 계세요.

사실 내용이 없어서 밀리는 거지
카페 자체가 문제가 아닌 거잖아요.

바꾸면 되는 포인트 정리해뒀어요.
2분이면 확인되는 자료인데 받아보실래요? :)
```
→ 왜 이 문장: "카페가 문제가 아니다"는 표현이 사장님의 자존심을 지켜주면서 문제 원인을 플레이스 관리 부재로 자연스럽게 돌리고, 해결 가능성을 열어준다.

```

      — [업체명] 맞춤 분석본

임팩트 한 줄:
"지금 이 순간도 손님이 1위 업체 쪽으로 넘어가고 있습니다."
```

### 슬라이드 2: 문제 인식

```
제목: 광고비는 나가는데, 왜 손님은 안 늘까요?

────────────────────────────────
경기 북부 소상공인 10곳 중 8곳이 겪는 현실

"열심히 하는데 왜 나만 손님이 없지?"
"SNS도 해봤는데 효과가 없었어요."
"대행사 썼다가 돈만 날렸어요."
────────────────────────────────

📊 실제 데이터로 보면:

네이버 플레이스 방문자 수 = 매출 변동의 82% 설명
(실증 데이터: 방문자 1명 = 평균 9,256원 매출)

하루 방문자 50명 차이
= 월 매출 1,400만원 차이

경기도 소상공인 중 디지털 광고 활용 비율: 단 15.4%
→ 100곳 중 85곳이 아직 시작도 안 했다는 뜻

문제는 마케팅이 아닙니다.
네이버 플레이스 하나를 제대로 안 잡고 있는 것입니다.
```
