# 리안 컴퍼니 — 경기 북부 소상공인 네이버 플레이스 DM 영업 완성 스크립트

---

## DM/문자 스크립트

### 1차 메시지 (3가지 버전)

---

**버전 A — 데이터 충격형 (미용실 1순위 타겟 / Cold 고객)**

```
사장님, 안녕하세요 😊
저 오늘 '의정부 미용실' 네이버에서 검색해봤거든요.

[업체명]이 지금 [N]위에 있는데
1~3위 업체들이랑 비교해보니까

한 달 방문자 수가 약 3~4배 차이 나더라고요.
이거 방문자 1명당 약 9,200원 매출 차이로 계산하면
매달 [N]만 원 정도가 윗 업체들한테 넘어가고 있는 거예요.

사장님 가게 기준으로 정확히 계산해서
무료 분석 자료 만들어뒀는데, 한번 받아보실래요?
부담은 전혀 없어요, 판단은 사장님이 하시면 됩니다.

— 리안 컴퍼니 드림
```

→ 왜 이 문장: "매달 [N]만 원이 넘어가고 있다"는 추상적 위기감이 아닌 데이터 기반 손실 계산이라 사기 패턴으로 인식되지 않고 신뢰를 유발한다.

---

**버전 B — 공감 선접촉형 (식당 타겟 / 바쁨·모름 Pain 직접 건드리기)**

```
사장님, 안녕하세요 😊
장사하시면서 네이버 플레이스 관리는
따로 신경 쓸 틈도 없으시죠?

사실 사진이랑 메뉴 열심히 올려도
순위가 안 오르는 게 당연한 거예요.
네이버가 사진 개수가 아니라
'방문자 체류시간'이랑 '외부 유입'을
같이 보거든요. 올리는 것만으론 부족해요.

오늘 '포천 맛집' 검색해봤는데
[업체명]이 지금 [N]위에 있어서
사장님 가게 맞춤 분석 자료 만들어뒀어요.

무료니까 부담 없이 한번 보시겠어요? 😊
```

→ 왜 이 문장: "사진 꼼꼼히 올렸는데 순위 안 변한다"는 실제 Pain 언어를 그대로 인정해줌으로써 '이 사람은 나를 이해한다'는 첫 신뢰를 형성한다.

---

**버전 C — 궁금증 유발형 (카페 타겟 / 낮은 디지털 인식 고객용)**

```
사장님, 안녕하세요 😊
갑자기 연락드려서 죄송한데요,

오늘 '양주 카페' 네이버 검색하다가
[업체명] 보고 연락드렸어요.

혹시 지금 플레이스 순위가
왜 안 오르는지 궁금하신 적 있으세요?

대부분 사장님들이
"뭘 어떻게 해야 하는지 모르겠다"고 하시더라고요.
그게 당연한 거예요, 네이버 로직이 생각보다 복잡해서요.

사장님 가게 기준으로 딱 한 장짜리
무료 분석 자료 만들어뒀는데
한번 보내드려도 될까요? 😊
```

→ 왜 이 문장: 디지털 인식이 낮은 경기 북부 타겟에게 "모르는 게 당연하다"는 면죄부를 주면서 호기심을 자극해 거부감 없이 응답을 유도한다.

### 2차 메시지 (신뢰 형성 + 가치 증명)

*응답 후 30분 이내 발송 — 무료 진단 리포트와 함께*

```
사장님, 관심 가져주셔서 감사해요 😊
말씀드린 분석 자료 바로 보내드릴게요!

────────────────
📊 [업체명] 플레이스 무료 진단 리포트
────────────────

📍 현재 순위
'[지역] [업종]' 키워드 기준 → 현재 [N]위
(이 키워드 월간 검색량: 약 [N]회)

📊 경쟁 업체 비교 (상위 3개 평균)
• 리뷰 수: 경쟁 평균 [N]개 vs 사장님 가게 [N]개
• 사진 수: 경쟁 평균 [N]장 vs 사장님 가게 [N]장
• 최근 소식 업데이트: 경쟁사 주 [N]회 vs 사장님 가게 [N]일 전 마지막

💸 지금 발생하고 있는 기회비용 계산
현재 순위([N]위) vs 3위 진입 시 추정 방문자 차이
→ 월 약 [N]명 차이
→ 방문자 1명당 매출 기여 약 9,200원 기준
→ 매달 약 [N]만 원이 윗 순위 업체로 넘어가고 있어요

────────────────

사장님 가게가 3위 안에 들어오면
지금 이 기회비용이 그대로 사장님 매출이 돼요.

어렵거나 복잡한 거 없고요,
실제로 저희가 비슷한 조건의 [지역] [업종]을
[N]개월 만에 [N]위 → [N]위로 올린 적 있거든요.

궁금하신 거 있으시면 편하게 여쭤보세요 😊
```

### 3차 메시지 (제안 + 클로징)

*2차 메시지 발송 후 1~2일 내, 또는 리포트 확인 반응 후*

```
사장님, 리포트 보시고 어떠셨어요? 😊

솔직히 말씀드리면요,
저희가 사장님 가게 처음 분석했을 때
"이 정도면 금방 올릴 수 있겠다" 싶었어요.
기본기가 있는데 알려준 사람이 없었던 거거든요.

저희가 실제로 하는 일은 딱 세 가지예요.

✅ 체류시간 올리는 콘텐츠 세팅
   (어떤 사진·글이 사람들이 오래 보는지 저희가 직접 올려요)

✅ 외부 유입 만들기
   (SNS·맘카페에서 플레이스로 들어오는 경로 구축)

✅ 리뷰 관리 + 소식 정기 업데이트
   (사장님은 장사에만 집중하시면 돼요)

━━━━━━━━━━━━━━━━━━

📦 플레이스 집중 케어 패키지

• 기간: 3개월 (최소 성과 확인 기간)
• 월 비용: [N]만 원
• 포함 내역:
  - 플레이스 전체 세팅 최적화 1회
  - 월 소식 [N]회 대신 작성·업로드
  - 사진 기획·업로드 월 [N]컷
  - 외부 유입 채널 연동 세팅
  - 월 순위·방문자 리포트 1회 제공

━━━━━━━━━━━━━━━━━━

아까 계산해드린 기회비용이 월 [N]만 원이었잖아요.
서비스 비용이 그 절반도 안 돼요.
3개월 후 3위 안에 못 들어오면
그건 저희가 책임질 문제예요.

부담되시면 1개월 체험도 가능해요.
한번 해보시겠어요? 😊
```

### 4차 메시지 (마감 압박)

*3차 메시지 후 2~3일 무응답 시*

```
사장님, 혹시 아직 고민 중이세요? 😊

사실 제가 한 번에 여러 업체를 맡으면
각 가게에 신경을 덜 쓰게 되거든요.
그래서 경기 북부 지역은
업종별로 [N]곳까지만 함께하고 있어요.

지금 의정부 미용실 쪽에
자리가 딱 [N]곳 남아 있어요.

자리 차면 저도 새로 받기가 어렵고
사장님도 다음 시즌까지 기다리셔야 해서요.

이번 주 안으로 결정 주시면
첫 달 세팅비 없이 진행해드릴 수 있어요.

한번 믿어봐 주실 수 있으세요? 😊
```

### 재발송 메시지 (3일 후 무응답 시)

```
사장님, 안녕하세요 😊
며칠 전에 플레이스 분석 자료 보내드렸던
리안이에요, 혹시 기억하실지 모르겠네요.

바쁘실 때 연락드린 것 같아서
한 번 더 여쭤봐도 될까 해서요.

자료 아직 못 보셨으면 다시 보내드릴게요.
보셨는데 관심 없으시면 그렇다고 말씀해 주셔도 돼요,
그럼 더 연락 안 드릴게요 😊

판단은 항상 사장님이 하시는 거니까요.
```

**임팩트 한 줄:**
> "열심히 하는데 순위가 안 오른다면, 방법이 틀린 게 아니라 로직을 모르는 겁니다."

### 슬라이드 2: 문제 인식

**제목:** 사장님만 모르는 게 아닙니다 — 경기 북부 소상공인 현실

```
📌 이런 상황, 사장님도 겪고 계신가요?

"사진이랑 메뉴 꼼꼼히 올렸는데 순위가 안 바뀌어요"
"마케팅 신경 쓰고 싶은데 장사하다 보면 하루가 끝나요"
"대행사 연락은 오는데 다 비싸거나 믿을 수가 없어요"

━━━━━━━━━━━━━━━━━━━━━━

📊 실제 데이터로 본 현실

• 소비자 90%가 가게 찾을 때 네이버 검색 사용
• 플레이스 1~3위 업체가 전체 클릭의 70% 이상 독식
• 플레이스 방문자 1명 증가 = 매출 약 9,200원 증가
  (네이버 플레이스-매출 상관계수 0.91, 실증 데이터 기반)

• 경기 북부 소상공인 디지털 마케팅 역량 평균: 41.1점 / 100점
• 음식점 업종 평균: 38.1점 — 전 업종 중 최저

━━━━━━━━━━━━━━━━━━━━━━

💡 결론:
순위가 안 오르는 건 사장님이 게으른 게 아닙니다.
네이버가 사진 개수가 아니라
체류시간·외부유입·전환율을 함께 보기 때문입니다.
이걸 모르는 게 당연합니다.
```
