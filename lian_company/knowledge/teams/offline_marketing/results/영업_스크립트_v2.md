## DM/문자 스크립트

### 1차 메시지 (3가지 버전)

**버전 A — 손실 수치 충격 (전환율 최고, 1순위 타겟용)**
```
사장님, [업체명] 이번달 신규 고객 27명 놓치고 계세요.

'[강남구] 미용실' 검색 시 [경쟁업체명]이 1위 (월 431회 노출)
사장님은 9위 (월 23회)

8위 차이 = 신규 고객 월 27명 차이
(1명당 평균 50만원 = 월 1,350만원 매출 손실)

72시간 내 1-3위 올리는 법 알려드릴게요.
내일 오전 10시 or 오후 2시 중 5분 통화 가능하세요?

- 리안컴퍼니 마케팅팀 [담당자명] / 010-XXXX-XXXX
```
*전략: 첫 20자에 구체적 손실 수치 → 경쟁업체 실명으로 관계 손실 프레임 → 72시간 긴급성 → A/B 선택지로 가정적 클로징*

**버전 B — 단골 이탈 불안 자극 (관계 민감형 사장님용)**
```
[업체명] 사장님, 단골 손님들이 사장님 찾다가 [경쟁업체] 갔어요.

네이버에 '[지역] 미용실' 검색하면
사장님 업체 9위 → 화면에 안 보임
[경쟁업체] 1위 → 클릭 70% 독식

이거 계속되면 단골도 "여기 없어진 건가?" 생각합니다.
이번 주 안에 1-3위 올리는 법, 5분만 통화 가능하세요?

📞 내일 10시/14시 중 편한 시간 알려주세요.
리안컴퍼니 [담당자명] / 010-XXXX-XXXX
```
*전략: 단골 이탈 시나리오로 관계 손실 극대화 → "없어진 건가" 대사로 공포 자극 → 이번 주 마감으로 긴급성*

**버전 C — 경쟁 격차 시각화 (데이터 신뢰형 사장님용)**
```
[업체명] 사장님, 경쟁 격차 확인하셨어요?

📊 네이버 '[지역] 미용실' 검색 순위
1위 [경쟁업체] - 월 신규 예약 47건
9위 사장님 업체 - 월 신규 예약 3건

순위 8칸 차이 = 신규 손님 44명 차이
= 월 2,200만원 매출 차이

지금 안 올리면 격차는 더 벌어집니다.
72시간 안에 1-3위 올리는 법 → 5분 통화 가능하세요?

리안컴퍼니 [담당자명] / 010-XXXX-XXXX
수신거부 http://xxxxx
```
*전략: 숫자 시각화(📊)로 신뢰감 → 44명이라는 극명한 격차 → "더 벌어진다" 미래 손실 → 수신거부로 신뢰 시그널*

### 2차 메시지 (진단 결과 발송 — 통화 안 받을 시 24시간 후)

```
사장님, 어제 말씀드린 [업체명] 상세 분석 결과입니다.

📌 현재 상태 (치명적 3가지)
1. 검색 순위 9위 → 화면 노출 0%
2. 사진 5장 (경쟁업체 평균 18장)
3. 최근 리뷰 17개 (1위 업체 63개)

📉 손실 계산
- 순위 1위 vs 9위 = 월 신규 고객 44명 차이
- 1년 누적 손실 = 528명 x 50만원 = 2억 6,400만원

📈 개선 시 예상 효과 (72시간 내 조치 시)
✅ 1-3위 진입 → 월 신규 고객 +40명
✅ 사진/리뷰 최적화 → 예약 전환율 +35%
✅ 예상 매출 증가 월 +1,800만원

이번 주 금요일까지 계약 시에만 
'경쟁업체 약점 분석 리포트(30만원 상당)' 무료 제공합니다.

5분 통화 or 방문 상담 원하시면
"통화" or "방문" 중 하나만 답장 주세요.

리안컴퍼니 [담당자명] / 010-XXXX-XXXX
```
*전략: 진단 결과 형식으로 전문성 → 치명적 3가지로 문제 임팩트 → 2억 손실로 Implication 확대 → 금요일 마감 + 경품으로 희소성*

```
✔ 경쟁업체 약점 분석 리포트 (30만원)
✔ 인스타그램 연동 세팅 (20만원)

📞 리안컴퍼니 [담당자명] / 010-XXXX-XXXX

```
*전략: 마감 2자리로 희소성 → 패키지 구체화로 실체감 → 환불 보장으로 리스크 제거 → ROI 계산(1명=본전) → 금요일 마감 압박*

```
[경쟁업체명] 1위 유지 중
```

### 슬라이드 2: 문제 인식 (충격)
```

신규 고객이 경쟁업체로 가고 있습니다.

[숫자 박스]
📊 '[지역] 미용실' 검색 결과 (최근 30일)

1위 [경쟁업체명]
- 검색 노출 431회
- 클릭 289회 (클릭률 67%)
- 예약 전환 47건
→ 월 매출 약 2,350만원

9위 [업체명] (사장님)
- 검색 노출 23회
- 클릭 8회 (화면에 거의 안 보임)
- 예약 전환 3건
→ 월 매출 약 150만원

[충격 문구]
순위 8칸 차이 = 신규 고객 44명 차이
= 월 2,200만원 매출 차이
= 연간 2억 6,400만원 손실

사장님은 지금 이 돈을 경쟁업체에 주고 계십니다.
```

### 슬라이드 3: 현재 상황 진단
```
[업체명] 네이버 플레이스 현황 진단

❌ 치명적 문제 3가지

1️⃣ 검색 순위: 9위 → 11위 (하락 중)
   [막대그래프] 
   1위 노출: 67% / 9위 노출: 0.8%

2️⃣ 콘텐츠 빈약
   사진: 5장 (경쟁업체 평균 18장)
   리뷰: 17개 (1위 업체 63개)
   → 들어와도 안 믿고 나감

3️⃣ 키워드 최적화 0%
   현재 노출 키워드: 3개
   놓치는 키워드: 17개
   (예: [지역] 염색, [지역] 펌, [지역] 여자 미용실 등)

[진단 결과]
현재 상태 지속 시 3개월 내 15위 밖 이탈 예상
→ 신규 고객 유입 사실상 0명
→ 단골 이탈 시작 (검색해도 안 나오니 "폐업한 줄")
```
