- 핵심 가치: "대충 이런 거 만들고 싶어" → 5분 뒤 Claude Code 복붙용 
  설계서 + 마케팅 플랜 + 성과 개선 루프까지 자동 완성

- 목적별 자동 최적화:
  - 매출 극대화 → 가격 전략 + 수익모델 + 광고 설계
  - 바이럴 극대화 → 채널 선정 + 카피 A/B + 커뮤니티 전략  
  - 생산성 극대화 → 자동화 플로우 + 툴 조합 추천
