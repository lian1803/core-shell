# /save — 세션 자동 저장

이 명령어가 실행되면 아래를 즉시 수행해라. 질문하지 마라.

---

## Step 1: 이번 대화에서 한 것 파악

대화 내용을 전부 돌아보고 다음을 추출해라:
- 어떤 프로젝트를 작업했는가
- 어떤 파일을 만들었거나 수정했는가
- 어떤 테스트를 실행했고 결과는 무엇인가
- 어떤 결정을 내렸는가
- 다음에 해야 할 것은 무엇인가

---

## Step 2: memory/ 파일 업데이트

`C:\Users\lian1\.claude\projects\C--Users-lian1-Documents-Work-LIANCP\memory\` 에서:

1. 관련된 기존 memory 파일 찾아서 업데이트
   - project_*.md — 프로젝트 현황 변경사항 반영
   - feedback_*.md — 새로운 피드백/수정 사항 반영
   - user_*.md — 유저 정보 변경 반영

2. 새로 알게 된 중요한 것이 있으면 새 파일 생성

3. MEMORY.md 인덱스 업데이트

---

## Step 3: STATUS.md 업데이트

`C:\Users\lian1\Documents\Work\LIANCP\STATUS.md` 에:

```
## 마지막 세션 (YYYY-MM-DD)
- 작업한 프로젝트: [프로젝트명]
- 한 것: [구체적으로]
- 결과: [수치/상태]
- 변경한 파일: [목록]
- 다음에 할 것: [구체적으로]
```

```

```
