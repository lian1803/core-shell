## 이슈별 합의 사항

### 이슈 1: 가격 정책

#### 논의 내용
- **CTO 분석:** DALL-E 3 이미지 1장 $0.04 + GPT-4o 캡션 $0.005 = 일 $0.045
- **월간 AI 비용:** 30일 x $0.045 = $1.35 (약 1,800원/유저)
- **19,900원 기준 마진:** 18,100원 (90.9%)

#### 결론
**BASIC 19,900원 유지.** 마진 구조 건전함 확인됨.

#### 플랜 구조 확정
| 플랜 | 가격 | 콘텐츠 | 이미지 | 재생성 |
|------|------|--------|--------|--------|
| BASIC | 19,900원/월 | 30회/월 | 포함 | 3회/일 |
| PRO | 39,900원/월 | 60회/월 | 포함 | 무제한 |

### 이슈 2: 무료체험 기간

#### 논의 내용
- **CPO 입장:** 30일이 아하 모먼트 도달에 충분. 전환율 극대화
- **CTO 우려:** 1,000명 체험 시 180만원 선투자 → BEP까지 1개월 소요
- **절충안 검토:** 14일 기본 + 인센티브 연장

#### 결론
**14일 기본 무료체험 + 조건부 7일 연장**

조건: 인스타그램에 instauto 리뷰 게시물 업로드 시 7일 추가 (실질 21일)

#### 이점
- AI 비용 절반 감소 (90만원/1,000명)
- 바이럴 마케팅 효과 (리뷰 게시물)
- 진성 유저 필터링

---

### 이슈 3: DALL-E 3 이미지 생성 정책

#### 논의 내용
- **CPO 입장:** 이미지가 핵심 차별화. 없으면 "캡션 생성기"로 전락
- **CTO 우려:** 이미지가 AI 비용의 90% 차지. PRO 전용 검토 권장
- **사용자 가치 분석:** 카페 사장님 Pain은 "사진 촬영 어려움" → 이미지 생성이 핵심

#### 결론
**전 플랜(BASIC, PRO) 모두 이미지 생성 포함**

이유:
1. 이미지 없는 서비스는 경쟁력 없음
2. 19,900원 가격에 이미지 포함해도 마진 90% 유지
3. PRO는 횟수 차별화로 업셀링 유도

### 이슈 4: Meta 앱 심사 대응 전략

#### 논의 내용
- **CTO 정보:** Meta Business App Review 2~4주 소요
- **리스크:** 심사 대기 중 출시 지연 시 경쟁사에 선점당할 위험
- **대안:** "수동 복사" 모드로 선출시 후, 심사 완료 시 자동 발행 업데이트

#### 결론
**Phase 1.0: 수동 복사 모드 선출시**

#### 출시 로드맵
| Phase | 기능 | 예상 시점 |
|-------|------|----------|
| 1.0 | 캡션+이미지 생성 → 복사 버튼 → 사용자가 인스타에 수동 붙여넣기 | MVP 출시 즉시 |
| 1.1 | Meta 심사 완료 후 자동 발행 기능 추가 | 출시 후 2~4주 |

#### UX 처리
- "복사하기" 버튼 클릭 시 캡션 클립보드 복사 + 이미지 다운로드
- "인스타그램 열기" 버튼으로 앱 딥링크 제공
- 자동 발행 대기 안내 배너: "곧 원클릭 자동 발행이 추가됩니다!"


## 개발팀 지시사항

### FE (프론트엔드)

1. **복사 기능 우선 구현**
   - 캡션 복사 버튼: `navigator.clipboard.writeText()`
   - 이미지 다운로드 버튼: Blob 다운로드 구현
   - 인스타그램 딥링크: `instagram://` 스킴 사용

2. **자동 발행 UI는 "준비 중" 상태로**
   - 발행 버튼 영역에 "자동 발행 준비 중" 배너 표시
   - Phase 1.1에서 활성화 예정

3. **체험 연장 UI 추가**
   - 14일 남았을 때 "리뷰 작성하고 7일 연장받기" 모달
   - 리뷰 인증 플로우 (URL 입력 → 검증)

### BE (백엔드)

1. **Prisma 스키마 수정**
   ```prisma
   model Subscription {
     // 기존 필드 유지
     trialEndAt          DateTime    // 기본 14일로 변경
     trialExtended       Boolean     @default(false)  // 연장 여부
     trialExtendedAt     DateTime?   // 연장 일시
     reviewUrl           String?     // 리뷰 게시물 URL
   }
   ```

2. **체험 연장 API**
   - `POST /api/trial/extend` - 리뷰 URL 검증 후 7일 연장
   - Instagram oEmbed API로 게시물 존재 여부 확인

3. **Instagram 발행 API는 Phase 1.1**
   - `/api/instagram/*` 라우트는 구조만 생성, 로직은 Phase 1.1
   - `/api/publish/now` → Phase 1.0에서는 404 반환

4. **콘텐츠 생성 제한 로직**
   - BASIC: 월 30회, 일 재생성 3회
   - 제한 초과 시 업그레이드 안내 모달 트리거

### 공통

1. **환경변수 (Phase 1.0)**
   ```
   OPENAI_API_KEY=        # GPT-4o + DALL-E 3
   SUPABASE_URL=
   SUPABASE_ANON_KEY=
   SUPABASE_SERVICE_KEY=
   TOSS_CLIENT_KEY=
   TOSS_SECRET_KEY=
   RESEND_API_KEY=
   ```

2. **환경변수 (Phase 1.1 추가)**
   ```
   META_APP_ID=
   META_APP_SECRET=
   INSTAGRAM_REDIRECT_URI=
   ```
