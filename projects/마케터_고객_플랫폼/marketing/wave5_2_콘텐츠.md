고객은 링크 클릭 → 카카오페이 또는 토스로 바로 결제.
결제 완료되면 제 채팅창에 즉시 알림.

무엇보다 **지금 완전 무료**입니다.

나중에 유료 전환될 수 있다고 하는데 얼리어답터로 지금 쓰는 게 이득인 것 같아요.

### 포스팅 2 — 상세 사용 후기

**제목**: ChatPay 실제 사용해보니 (채팅 중 결제 연동 툴 솔직 후기)

---

저번 포스팅에서 ChatPay를 간략히 소개했는데, 이번엔 실제 사용 과정을 스크린샷과 함께 정리해볼게요.

**설치 방법**
1. chatpay.kr 무료 가입
2. 워크스페이스 만들기
3. 스크립트 코드 복사 → 내 사이트 head에 붙여넣기
→ 끝. 진짜 5분이면 됩니다.

**실제 사용 시나리오**

고객이 위젯을 통해 채팅을 시작하면, 제 대시보드에서 바로 확인됩니다.
대화 중에 상품명과 금액 입력 → "결제 링크 전송" 클릭.
고객 채팅창에 결제 버블이 뜨고, 클릭하면 카카오페이 결제창 오픈.
결제 완료되면 제 대시보드에서 "결제 완료" 알림.

**솔직한 단점**
- 아직 베타라 UI가 심플한 편
- 카카오페이/토스 API Key 등록이 필요 (PG 가맹점 신청 선행 필요)

**결론**: 인스타 DM이나 카카오채널로 주문받는 분들께 특히 강추.
지금 무료니까 일단 써보시길!

## 채널 2: 인스타그램 릴스

### 릴스 1 — "전과 후" 비교

**영상 대본** (30초)

```
[0-5초] 자막: "인스타 DM 주문받고 이렇게 하고 계신가요?"
화면: 계좌번호 복붙하는 DM 캡처 (흔들리는 손 클로즈업)

[5-10초] 자막: "고객: 결제할게요! / 나: 계좌번호는..."
화면: 길고 복잡한 계좌이체 과정 재현

[10-15초] 자막: "결제 확인하느라 3시간..."
화면: 핸드폰 들고 입금 체크하는 장면

[15-22초] 자막: "ChatPay로 바꾼 후"
화면: ChatPay 대시보드에서 결제 링크 전송 → 고객 결제 → "결제 완료" 알림 화면

[22-28초] 자막: "채팅 중에 결제 링크 바로 전송"
화면: 결제 버블 UI 클로즈업

[28-30초] 자막: "지금 완전 무료. chatpay.kr"
```
