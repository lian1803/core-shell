## 3일 주기 체크리스트

### D+3 체크리스트
- [ ] 네이버 블로그 조회수 확인 (목표: 100+ / 미달 시 제목 수정)
- [ ] 인스타 릴스 조회수 확인 (목표: 300+ / 미달 시 해시태그 추가)
- [ ] 오픈채팅 반응 확인 (댓글/질문 대응)
- [ ] chatpay.kr 가입자 수 확인
- [ ] **Action**: 가장 반응 좋은 채널에 콘텐츠 추가 집중

### D+7 체크리스트
- [ ] 전체 KPI 대비 달성률 계산
- [ ] 유입 채널별 전환율 비교
  - 가입 완료율: 가입 / 방문
  - 온보딩 완료율: 워크스페이스 생성 / 가입
- [ ] 가장 효과적인 채널 TOP 1 선정 → 리소스 집중
- [ ] **Action**: 미달 채널 전략 수정 or 포기

### D+14 체크리스트
- [ ] 워드오브마우스 확인 (구글 서치콘솔, 언급 검색)
- [ ] 사용자 인터뷰 요청 (초기 가입자 3명 DM)
- [ ] 실사용 후기 콘텐츠 수집
- [ ] **Action**: 후기 기반 콘텐츠 제작 (Social Proof 강화)

## 전환 퍼널 최적화 포인트

```
방문 → 가입 → 워크스페이스 생성 → 위젯 설치 → 첫 채팅 → 첫 결제 링크 발송
  ↓       ↓          ↓                  ↓           ↓              ↓
측정   이탈 파악   온보딩 개선        설치 가이드    채팅 UX        결제 UX
```

**이탈 지점별 대응**:
- 방문→가입 이탈: 랜딩페이지 개선 (무료 강조, 스크린샷 추가)
- 가입→워크스페이스 이탈: 온보딩 단계 줄이기
- 워크스페이스→위젯설치 이탈: 설치 가이드 영상 제공

## 성과 루프 자동화 계획

**3개월 후 유료 전환 시점 판단 기준**:
- 월 활성 마케터 500명 이상
- 월 결제 링크 발송 건수 2,000건 이상
- 결제 완료 전환율 30% 이상
→ 위 조건 달성 시 유료 플랜 도입 발표 (FREE_TRIAL 7일 부여 후 유료 전환)
