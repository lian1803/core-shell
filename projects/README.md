# projects/

리안컴퍼니가 설계 완료한 프로젝트들이 여기에 자동 생성됨.

## 구조

```
projects/
└── {프로젝트명}/
    ├── CLAUDE.md          ← 리안컴퍼니가 설계서 기반으로 자동 생성
    └── (UltraProduct가 여기서 코드/마케팅/분석 실행)
```

## 흐름

1. 리안: 아이디어 입력
2. 리안컴퍼니(이사팀): 조사 → 전략 → 검증 → 판단
3. 리안: "ㅇㅇ 해" 승인
4. 리안컴퍼니(지훈): 설계서 → `projects/{프로젝트명}/CLAUDE.md` 자동 생성
5. UltraProduct: CLAUDE.md 읽고 → 개발 + 마케팅 + 분석 + 루프 실행
